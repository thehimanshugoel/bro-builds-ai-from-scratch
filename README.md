# Bro Builds AI From Scratch

A bro's journey to build AI from scratch using pure Python — no TensorFlow, no PyTorch, no magic.

Bro teaches a computer AI concepts step by step, one chapter at a time. Each chapter is a conversation between Bro and Computer, and each chapter's code stands on its own.
